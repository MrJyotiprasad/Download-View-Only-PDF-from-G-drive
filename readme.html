📄 HAR to High-Quality PDF Converter

This project extracts high-quality base64-encoded images from a .har file (e.g., from browser DevTools) and converts them into a single high-resolution PDF. It automatically detects any .har file in the current folder and processes it.

🧰 Requirements

Before running the script, make sure you have the following installed:

✅ Python Modules

  • tqdm – for progress bars
  • pillow – for image processing and PDF conversion

🔧 Install Dependencies

pip install -r requirements.txt

🚀 How to Use

📁 Folder Structure

view-only-pdf-downloader/ ├── main.py
├── requirements.txt
├── readme.txt
└── README.MD

▶️ Step-by-Step

  1. Place your .har file (exported from browser DevTools) into the view-only-pdf-downloader/ folder.
  2. Open a terminal in that folder.
  3. Run the script:
python main.py

The script will:

  • Automatically find the first .har file in the folder
  • Extract all base64-encoded images
  • Save them in the extracted_images/ folder
  • Convert them into a high-quality PDF (final_high_quality_output.pdf)
  • Prompt you to either keep or delete the extracted images

📝 Notes

  • Only image/* MIME type with base64 encoding is processed
  • Images are saved with their correct extensions (e.g., .png, .jpg)
  • Output PDF is 300 DPI – suitable for printing
  • You will be prompted after PDF generation to clean up image files
